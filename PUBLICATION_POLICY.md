# Public Documentation Policy

This repository publishes a public implementation and public documentation for the OpenWave Identity Registry.

Rules:

- Do not commit internal continuation or workspace-context files.
- Treat committed docs output as public source material.
- Run `./scripts/check-public-surface.sh` before publishing docs updates.
- Keep identity focused on alias ownership and routing facts. Do not turn the registry into a payment-session or presentment state store.
